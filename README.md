1. [pypi](https://pypi.org/project/py2neo/)

## Commands

1. from clean install without pipenv
1. ```shell
   > python install pipenv Flask Graphene py2neo
   ```

1. with pipenv installed already
1. ```shell
   > pipenv shell
   > pipenv install
   > flask run
   ```
